# Major-TI Question Bank v3

适用页面：
- `v3.html`

编辑规则：
- 一道题就是一个 `## Q编号` 区块。
- `meta` 表当前只需要 `type`，固定为 `choice`。
- 题干放在 `text` 表。
- 选项表使用 `text` 和 `score` 两列。
- `score` 写成 `a:1` / `b:1` / `c:1` / `d:1`
- 改完后运行：`node sync-questions.js`
